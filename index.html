Inlämningsuppgift 1

För betyget Godkänt:

  1. Skapa ett nytt projekt med en html fil som har minst 3 element varav ett av dessa element är en bild (img).
  2. Skapa en .css fil och importera den till den html template du tidigare skapat
  3. I din .css skall du använda dig av CSS Grid för att ge ditt projekt ett responsivt beteende.

Krav: Sidan skall vara responsiv till både Desktop samt Mobile och detta skall åstadkommas med CSS Grid och Media Queries.
